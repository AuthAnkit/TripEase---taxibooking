🚀 Built my first backend project — *TripEase (Cab Booking System)*

I just completed a **Spring Boot backend project** where users can book cabs and get real-time booking confirmation via email.

🔧 **Tech Stack:**
• Java + Spring Boot
• Spring Data JPA (Hibernate)
• MySQL
• JavaMailSender (Email Integration)
• Swagger (API Testing)

💡 **Key Features:**
• Customer can book a cab using REST API
• System automatically assigns an available cab
• Fare calculation based on distance × per km rate
• Cab availability management (no double booking)
• Email confirmation sent instantly after booking

⚙️ **Architecture:**
Controller → Service → Repository → Database

* DTO & Transformer layer for clean code structure

📌 **What I learned:**
• Writing clean layered backend architecture
• Handling entity relationships (OneToMany, OneToOne)
• Building real-world business logic (like Uber flow)
• Integrating email services in backend

This is just the beginning — next step is improving it with:
👉 Better cab selection (nearest driver logic)
👉 Concurrency handling (real-world scaling)

Would love feedback and suggestions 🙌
